# 📄 README.md

## 🎮 Nombre del Proyecto
Juego: Piedra, Papel o Tijera

## 🎯 Objetivo
Desarrollar un programa interactivo en lenguaje Python que permita al usuario jugar al clásico juego de "Piedra, Papel o Tijera" contra la computadora, aplicando conceptos de programación como variables, estructuras de control, funciones, manejo de errores y lógica condicional.

## 📋 Descripción de las Funcionalidades
El programa cuenta con las siguientes características principales:

1. **Iniciar juego y mostrar menú principal**
   - Presenta las opciones disponibles para el usuario de forma clara y organizada.

2. **Elegir jugada**
   - Permite al jugador seleccionar entre las opciones: Piedra, Papel o Tijera.
   - Valida que la entrada sea correcta, evitando errores por caracteres no permitidos o valores fuera de rango.

3. **Generar jugada de la computadora**
   - La computadora elige su opción de forma aleatoria mediante la librería `random`.

4. **Comparar jugadas y definir ganador**
   - Evalúa las reglas del juego para determinar si gana el usuario, la computadora o hay empate.
   - Actualiza automáticamente la puntuación según el resultado obtenido.

5. **Consultar marcador**
   - Muestra en cualquier momento la puntuación acumulada tanto del jugador como de la computadora.

6. **Salir del juego**
   - Finaliza la ejecución del programa de forma ordenada.
